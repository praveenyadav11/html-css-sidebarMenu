# Sidebar Menu Project

A responsive toggleable sidebar navigation menu built using pure HTML and CSS, featuring smooth slide animations and Font Awesome icons.

## Features

- Toggle sidebar open/close using checkbox hack (no JavaScript)
- Smooth slide-in/slide-out animation with CSS transitions
- Hamburger menu icon to open and close (X) icon to close
- Navigation links with icons (Gallery, Shortcuts, Exhibits, Events, Store, Contact, Feedback)
- Social media icons section (Facebook, Twitter, YouTube, Instagram) with hover effects
- Full-screen background image
- Hover effects on menu items and icons

## Technologies Used

- HTML5
- CSS3
- Font Awesome 7.0.1 (via CDN)

## Setup

1. Clone or download this repository
2. Add your own background image and name it `photo.jpg` (or update the path in `style1.css`)
3. Open `index.html` in a browser

## How It Works

The sidebar toggle is implemented using the CSS "checkbox hack" — a hidden checkbox (`#check`) controls the sidebar's `left` position via the `:checked` pseudo-class, eliminating the need for JavaScript.
